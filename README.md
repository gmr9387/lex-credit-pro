# Credit Repair AI

An autonomous, AI-powered credit optimization platform that analyzes credit reports, generates FCRA-compliant dispute letters, and helps users rebuild their credit scores.

## Features

- **AI Credit Report Analysis** — Upload reports and get instant flagged-item detection
- **Automated Dispute Generation** — FCRA/FDCPA-compliant dispute letters with batch processing
- **Score Tracking & Simulation** — Monitor progress and simulate credit actions
- **Bureau Response Tracking** — Track responses and deadlines across all three bureaus
- **Goodwill Letter Generator** — AI-crafted goodwill letters for late payment removal
- **CFPB Escalation Wizard** — Guided escalation when bureaus fail to respond
- **Gamification & Achievements** — Stay motivated with milestones and badges
- **Credit Education Center** — Quizzes, guides, and AI mentoring

## Tech Stack

- React + TypeScript + Vite
- Tailwind CSS + shadcn/ui
- Supabase (Auth, Database, Edge Functions, Storage)
- Progressive Web App (PWA) support

## Getting Started

```sh
# Clone the repository
git clone <YOUR_GIT_URL>

# Navigate to the project directory
cd credit-repair-ai

# Install dependencies
npm install

# Start the development server
npm run dev
```

## Deployment

Build the production bundle:

```sh
npm run build
```

Deploy the `dist/` folder to any static hosting provider (Vercel, Netlify, Cloudflare Pages, etc.).
